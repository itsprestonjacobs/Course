# Welcome to the Discord Bot Course

By the end of this course you'll have built a real Discord bot in Python — one that posts
fancy embeds, runs moderation commands, and hands out support tickets with buttons. No
prior coding experience needed. We start from "what is Python" and go all the way to a bot
running in your own server.

## What you'll build

A single bot with three big features:

- **Embeds** — clean, colorful info messages instead of plain text.
- **Moderation** — `/kick`, `/ban`, `/timeout`, `/purge`, and `/warn`, all locked behind
  the right permissions.
- **A ticket system** — a button people click to open a private support channel, and a
  close button that saves a transcript.

## Before you start

- A computer (Windows, Mac, or Linux — screenshots here use Windows).
- A Discord account.
- About 2–3 hours total. Each lesson is bite-sized; take breaks.

## How to use this course

Read a lesson, type the code yourself (don't just copy — you learn by typing), and run it.
If something breaks, check the **Troubleshooting** page — almost every beginner hits the
same handful of errors, and they're all in there.
